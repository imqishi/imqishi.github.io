Leetcode 33

一个有序数组从某个位置左右换位，如：

求 target 是否在该数组中，若有返回位置，否则返回 -1

要求时间复杂度 O(logn)

思路

看到这个时间复杂度和部分有序就可以想到是二分搜索的变形，那么如何进行二分搜索呢？本题的关键就是对部分有序数组与 mid 位置数字的情况划分：

第一种情况是可以正常用二分查找的时候，体现为目标值在原数组的左半部分，且中间位置也是在左半部分
第二种情况是目标值在数组的左半部分，而中间位置却在右半部分，此时一定向右找
第三种情况是目标值在数组的右半部分，而中间位置却在左半部分，此时一定向左找
